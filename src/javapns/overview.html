JavaPNS 2.1

A Java library for interacting with Apple's Push Notification Service.

Usage

The simplest way of pushing notifications with JavaPNS is to use the javapns.Push class:


import javapns.Push;
        
public class PushTest {
 
	public static void main(String[] args) {
	
		Push.alert("Hello World!", "keystore.p12", "keystore_password", false, "Your token");
	}
}
For more details about using the library, see the on-line wiki at: http://code.google.com/p/javapns/w/list
